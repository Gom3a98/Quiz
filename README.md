# Quiz
System to organize and manage the Quizes in our company

#installation Guide

1- Download & Install Xampp from this link >> https://www.apachefriends.org/index.html
2-pull the master branch in this path (C:\xampp\htdocs)
3-open the Downloaded folder and cut 'quizdb' folder to this path (C:\xampp\mysql\data)
4-after installing xampp open it and start apache server and mysql database
5-Finally open the browser and requist this URL ('http://localhost/Quiz')
